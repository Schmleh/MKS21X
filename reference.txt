GIT Commands!!!

clone <bleh>
makes the repository <bleh> in your local machine

add
adds the things
(makes them ready for commiting)
using with -A does it to all files in the repo

commit -m "<message>"
makes your ready to commit stuff into commits
you must put a message like:
"Call Pinkie Pie at 1-PIN-KHO-RSIE"

push
makes your commits into real public stuffs


pull
gets the real public stuffs
